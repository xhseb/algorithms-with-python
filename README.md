# algorithms-with-python

- 시간복잡도

[파이썬 기본 함수들의 시간복잡도](https://wiki.python.org/moin/TimeComplexity)

|빅오 표기법|명칭|
|------|---|
|O(1)|상수 시간(Constant time)|
|O(logN)|로그 시간(Log time)|
|O(N)|선형 시간|
|O(NlogN)|로그 선형 시간|
|O(N²)|이차 시간|
|O(N³)|삼차 시간|
|O(2ⁿ)|지수 시간|
